Security & Storage

Data Architecture & Privacy

Storage & Updates

Database visible to citizens via chatbot must be uploaded to our system.

Vector store, message history, and contacts reside in the iNua Cloud. All updates done directly in iNua systems.

Anonymous Feedback Limits

Data is tied to specific contacts and phone numbers. Consequently, questionnaire answers are not anonymous.

Opt-In Privacy & API Integration

Linking the system with social security IDs will be a progressive process via API. We do not hold highly sensitive data ourselves. Due to data sensitivity, this requires a data protection partner (deferred beyond pilot phase).
